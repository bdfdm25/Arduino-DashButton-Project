Arduino Dash Button

Project developed as a final project for the Computer Science courses at PUC-MG university.

E-commerce, or electronic commerce, is the online way (by the Internet) for buying or selling. By this new buying concept it is possible to acquire a big variety of products and services available in the internet through online shops. Internet of Things (IoT) is the network of physical devices, vehicles, buildings, and other items that have embedded technology, sensors and network connectivity capable of collect and exchange data. Aiming link these two concepts widely used nowadays came the idea to develop a prototype for a button, based in Arduino technology, which could be used to make it easy to purchase recurrent online orders, that is, those purchases that happens repeatedly. 
In this repository you will find two directories: Arduino & Web. In the first one will be the code responsible to make the button works using the Arduino board. In the second one will be the codes for the ecommerce website model (Directory: ecommerce) and for the configuration webpage used to set the WiFi data in the button (Directory: Config_Botao).

Arduino Dash Button:

Este projeto foi desenvolvido como trabalho de conclusão de curso para o curso de Ciência da Computação na PUC-MG. 

E-commerce, ou comércio eletrônico, é a forma online (pela Internet) de compra ou venda. Por meio deste novo conceito de compra é possível adquirir os mais diversos produtos e serviços disponíveis na internet através de lojas virtuais. Internet of Things (IoT), é uma rede de objetos físicos, veículos, prédios e outros objetos que possuem tecnologia embarcada, sensores e conexão com rede capaz de coletar e transmitir dados. Com o intuito de unir estes dois conceitos amplamente utilizados hoje em dia, surgiu- se a ideia de desenvolver o protótipo de um botão, utilizando da tecnologia do Arduino, para que fosse possível facilitar a emissão de pedidos de compra online recorrentes, ou seja, aqueles pedidos realizados repetidamente. 
Neste repositorio você encontrara dois diretorios: Arduino e Web. No primeiro esta o codigo responsavel pelo funcionamento do botao desenvolvido utilizando a plataforma Arduino. No segundo estao os codigos do site modelo ecommerce (Diretorio: ecommerce) desenvolvido, e da pagina web utilizada para configurar os dados de rede WiFi no botao (Diretorio: Config_Botao).
